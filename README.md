# CBS Banking System

A Core Banking System (CBS) web application built using:

## 🚀 Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: PostgreSQL

## 🔥 Features
- CIF Management
- KYC & Compliance
- Demand Operations
- Approval Workflow
- Role-based Access (Admin, Manager, Officer)
